1 Introduccion

Telegram Downloader es una aplicacion multiplataforma desarrollada en Flutter que te permite buscar y descargar archivos directamente desde tus canales de Telegram. La app se conecta a Telegram usando la libreria nativa TDLib, proporcionando una experiencia rapida y segura.

🔍

Busqueda avanzada

Busca archivos en todos tus canales habilitados con filtros de temporada y episodio.

⬇

Descargas en tiempo real

Progreso en vivo, velocidad, ETA y control completo de cada descarga.

📺

Gestion de canales

Activa o desactiva canales para personalizar tu experiencia de busqueda.

🎨

Tema claro y oscuro

Cambia entre modo claro y oscuro con un solo toque.

Plataformas soportadas

📱 Android 💻 Windows 🍎 macOS 🐧 Linux

Requisitos previos

  • Una cuenta de Telegram activa
  • Credenciales de API de Telegram (API ID y API Hash) obtenidas en my.telegram.org
  • Estar suscrito a canales de Telegram que compartan archivos

2 Instalacion y descarga

Los binarios precompilados estan disponibles en la pagina de GitHub Releases del proyecto. Cada actualizacion del codigo genera automaticamente nuevos binarios para las 4 plataformas.

Archivos disponibles

Plataforma Archivo Instrucciones
Android telegram-downloader-android-vX.X.X.apk Descarga el APK e instalalo (habilitar fuentes desconocidas)
Windows telegram-downloader-windows-vX.X.X.zip Descomprime y ejecuta el .exe
macOS telegram-downloader-macos-vX.X.X.zip Descomprime y arrastra a Aplicaciones
Linux telegram-downloader-linux-vX.X.X.zip Descomprime y ejecuta el binario
💡 Consejo

En macOS, la primera vez que abras la app puede que el sistema la bloquee. Ve a Ajustes del sistema > Privacidad y seguridad y pulsa "Abrir de todos modos".

Compilacion desde codigo fuente

Si prefieres compilar la app tu mismo, necesitaras Flutter SDK >= 3.2.0 instalado:

# Clonar el repositorio
git clone https://github.com/kno/telegram-downloader-app.git
cd telegram-downloader-app

# Instalar dependencias
flutter pub get

# Ejecutar en modo desarrollo
flutter run

# Compilar para tu plataforma
flutter build apk --release      # Android
flutter build windows --release   # Windows
flutter build macos --release     # macOS
flutter build linux --release     # Linux

Pantalla de Busqueda con resultados y filtros de temporada

Como buscar

  1. Escribe el termino de busqueda en el campo principal (ej: nombre de serie, pelicula, etc.)
  2. Opcionalmente, rellena los campos de Temporada y Episodio para filtrar resultados de series
  3. Pulsa el boton Buscar o la tecla Enter
  4. Los resultados aparecen debajo, mostrando hasta 50 resultados por pagina

Tarjetas de resultado

Cada resultado muestra:

  • Nombre del archivo: titulo completo
  • Tamano: peso del archivo (ej: 2.4 GB)
  • Descripcion: informacion adicional si esta disponible
  • Canal de origen: nombre del canal (clickable para abrirlo en Telegram)
  • Fecha: fecha de publicacion
  • Boton Descargar: inicia la descarga del archivo

Filtro de temporada y episodio

El filtro reconoce patrones comunes en nombres de archivos:

PatronEjemplo
S01E05Breaking.Bad.S01E05.1080p.mkv
1x05Breaking.Bad.1x05.mkv

Paginacion

Si hay mas de 50 resultados, aparecen botones de Anterior y Siguiente en la parte inferior para navegar entre paginas.

⚠ Nota

Solo se busca en los canales que tengas habilitados. Si no obtienes resultados, verifica la pantalla de Canales.

Eliminar una descarga

Al pulsar el boton de eliminar, aparecen tres opciones:

  • Solo quitar: elimina la descarga de la lista pero mantiene el archivo
  • + Borrar archivo: elimina la descarga y tambien borra el archivo del disco
  • Cancelar: vuelve a la vista normal sin hacer nada

Informacion de progreso

Las descargas activas muestran informacion detallada en tiempo real:

  • Porcentaje: progreso numerico y barra visual
  • Velocidad: velocidad actual de descarga (ej: 4.2 MB/s)
  • ETA: tiempo estimado restante (ej: 3m 22s)

Guardar archivos completados

Cuando una descarga se completa, puedes usar Guardar como para copiar el archivo a la ubicacion que prefieras. Se abrira un selector de archivos del sistema donde podras elegir la carpeta de destino.

💡 Carpeta por defecto

Puedes configurar una carpeta de descarga por defecto en los Ajustes para no tener que elegir la ubicacion cada vez.

Funciones principales

Habilitar/Deshabilitar canales

Cada canal tiene un interruptor (switch) que puedes activar o desactivar. Solo los canales activados se incluiran en las busquedas.

Filtrar canales

Usa el campo de busqueda en la parte superior para filtrar la lista por nombre. Util si tienes muchos canales.

Acciones masivas

  • Todos: habilita todos los canales de una vez
  • Ninguno: deshabilita todos los canales de una vez

Recargar canales

Pulsa el icono de recarga en la barra superior para volver a obtener la lista de canales desde Telegram. Esto es util si te has suscrito a un nuevo canal.

💡 Consejo

Los canales que muestran @nombre son canales publicos. Los que muestran ID: -100... son canales/grupos privados.

9 Preguntas frecuentes

❓ No encuentro resultados al buscar

Verifica que tienes canales habilitados en la pantalla de Canales. Tambien asegurate de que los canales a los que estas suscrito compartan el tipo de contenido que buscas.

❓ La app muestra "No hay canales" en la pantalla de Canales

Pulsa el boton de recarga (🔄) en la barra superior. Si es la primera vez, la app necesita obtener la lista de canales desde Telegram, lo cual puede tardar unos segundos.

❓ La descarga se queda en "En cola" y no avanza

Telegram tiene limites de descargas simultaneas. Espera a que se completen las descargas activas. Si persiste, intenta pausar y reanudar la descarga.

❓ Error de autenticacion al iniciar la app

Si tus credenciales API han cambiado o tu sesion ha expirado, ve a Ajustes y cierra sesion. Luego vuelve a completar el asistente de configuracion.

❓ En macOS la app no se abre

macOS puede bloquear aplicaciones descargadas de internet. Ve a Ajustes del sistema > Privacidad y seguridad y pulsa "Abrir de todos modos" junto al aviso sobre Telegram Downloader.

❓ ¿Donde se obtienen el API ID y API Hash?

  1. Ve a my.telegram.org
  2. Inicia sesion con tu numero de telefono
  3. Ve a API Development Tools
  4. Crea una nueva aplicacion (o usa la existente)
  5. Copia los valores de api_id y api_hash

❓ ¿Puedo usar la app en varias plataformas a la vez?

Si, pero cada instalacion necesita su propia autenticacion. Ten en cuenta que Telegram puede limitar el numero de sesiones activas simultaneas.

❓ ¿Donde se guardan los archivos descargados?

Por defecto, los archivos se almacenan en la carpeta interna de la aplicacion. Puedes configurar una carpeta de destino por defecto en los Ajustes, o usar "Guardar como" en cada descarga completada para elegir donde guardarla.
